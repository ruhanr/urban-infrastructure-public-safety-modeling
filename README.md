# Data Science Portfolio — Ruhan Rahman

This repository contains selected data science and machine learning projects focused on predictive modeling, data integration, and model evaluation using real-world datasets.

## Core Focus Areas
- Data Engineering & ETL workflows  
- Regression and Classification Modeling  
- Cross-Validation & Hyperparameter Optimization  
- Geospatial Data Analysis  
- Model Interpretability  

## Featured Projects

### Urban Infrastructure & Public Safety Modeling
Regression and classification models analyzing violent crime and pedestrian-vehicle accident hotspots across 239 Charlotte census tracts.  
Techniques: spatial joins, feature engineering, cross-validation, grid search.

### Income & Risk Classification Modeling
Comparative evaluation of classification models to predict income level and credit risk outcomes.  
Techniques: logistic regression, SVM, decision trees, structured model validation.

Each project folder contains a dedicated README and detailed project report explaining methodology, implementation, and findings.
